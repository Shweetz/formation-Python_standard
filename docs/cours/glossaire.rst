*********
Glossaire
*********

.. glossary::

    Instance
        Un objet. On parle d'instance de classe, il s'agit d'un objet de type
        de la classe.

    Itérable
        Une collection qui peut être parcourue élément par élément. La
        collection doit donc être ordonnée.

    Objet
        Représentation réelle d'un type de donnée. Voir : :doc:`objet`

    Programmation Déclarative
        Paradigme de programmation consistant à déclarer les données du
        problème, puis à demander au programme de le résoudre.

    Programmation Fonctionnelle
        Paradigme de programmation de type déclaratif qui considère le calcul
        en tant qu'évaluation de fonctions mathématiques.

    Programmation Impérative
        Paradigme de programmation décrivant les opérations en une séquence
        d'instructions.

    Programmation Orientée Objet
        Paradigme de programmation consistant en la définition et l’assemblage
        de briques logicielles appelées objets. Voir : :doc:`objet`

    Structure de contrôle
        Instruction particulière qui dévie le flot de contrôle avec des
        constructions comme les alternatives (`if`…), boucles (`for`, `while`),
        les appels de fonction mais aussi la gestion des exceptions.
